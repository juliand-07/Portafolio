Backend con spring-boot

Este proyecto es un prototipo que busca implementar los conocimientos del desarrollo de proyectos de backend usando java como lenguaje y desarrollarlo en su framework spring boot este backend busca cumplir las mismas funciones del backend del proyecto “event” aquí buscamos conectar el proyecto con la db, la conexión con el front mediante consultas https, la encriptación y validadcion de contraseñas usando bicript y autenticaciones con JWT.
