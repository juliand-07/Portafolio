Deskary

Deskary es un software diseñado para la presentación y el manejo de los libros de una biblioteca, este software tiene como propósito cumplir como una landing page que muestre a los visitantes los libros a disposición de la biblioteca contando con un buscador para filtrar el libro deseado por el visitante, de igual manera cuenta con una adminpage que cumple la función de llevar el registro de los libros prestados, permitiendo no solo llevar un registro de prestamos sino el registro de clientes de la biblioteca, agregar nuevo libro, este software en su mayoría fue desarrollado con node js usando angular como desarrollo fronted manejando conceptos como los son el manejo de observables, el uso de formularios reactivos, usando servicios como conectores al back con respuesta https y enviando y recibiendo información con formatos json, para el backend se uso nest js manejando controladores, servicios,dtos y entidades, de igual manera se implemento el uso de migraciones para el manejo de cambios en la db y usando Docker como contenedor par el uso de postgres y pgadmincomo gestor de DB.
